# Anime List API

A modular and lightweight REST API built with FastAPI and MongoDB Atlas, optimized for mobile application consumption.

## 🚀 Quick Start

1. Install Dependencies:
uv sync --refresh

2. Set Environment Variables:
Create a .env file in the root directory:
MONGO_URI="your_mongodb_atlas_uri"

3. Run the Server:
uvicorn main:app --reload

Go to http://127.0.0.1:8000/docs to view the interactive API documentation.
